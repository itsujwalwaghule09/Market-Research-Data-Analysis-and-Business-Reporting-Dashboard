# Market Research Industry Analytics Dashboard (Power BI)

## Project Overview

This project is a comprehensive **Market Research Industry Analytics Dashboard** developed using **Microsoft Power BI**. The dashboard is built on a multi-table dataset containing market research operations, client management, survey responses, market sizing, forecasting, competitor analysis, campaigns, web analytics, and project performance data.

The objective of this project is to transform raw business data into actionable insights that help research firms, analysts, and decision-makers monitor performance, identify market opportunities, evaluate competitors, and track research activities through interactive visualizations.

---

## Business Problem

Market research organizations handle large volumes of data across multiple business functions, including:

* Client and project management
* Market sizing and forecasting
* Survey and respondent analysis
* Competitor tracking
* Research report performance
* Marketing campaign effectiveness
* Website engagement analytics

Managing these datasets separately makes it difficult to obtain a consolidated view of business performance. This dashboard addresses that challenge by integrating data from multiple sources into a single analytical platform.

## Power BI Development Process

### Data Preparation

* Imported and transformed Excel data using Power Query.
* Cleaned and standardized fields.
* Managed missing and inconsistent values.
* Created calculated columns and measures.

### Data Modeling

* Established relationships across 26 tables.
* Implemented a star-schema-inspired model.
* Optimized model performance for reporting.

### DAX Measures

Created custom DAX measures for:

* Revenue calculations
* Growth percentages
* Conversion rates
* Market forecasting KPIs
* Performance indicators
* Dynamic filtering and aggregation

### Visualization Design

Implemented interactive visualizations including:

* KPI Cards
* Bar Charts
* Line Charts
* Area Charts
* Donut Charts
* Matrix Tables
* Slicers
* Drill-through Pages
* Dynamic Filters
